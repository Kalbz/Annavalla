# Annavalla

Simple static website for my moms "Airbnb" Annavalla.

The site exists mainly to give guests a clean landing page with basic information and direct links to the booking platforms.

Live site:  
https://annavalla.se

## Purpose

This repository contains a very small static website used for:

- Showing information about the property
- Displaying photos
- Linking visitors to the Booking.com listing
- Providing a simple entry point for guests

The goal was to keep everything extremely lightweight and easy to maintain.

## Tech

The site is intentionally simple:

- HTML
- CSS
- Vanilla JavaScript
